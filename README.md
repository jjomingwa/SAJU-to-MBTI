# SAJU-to-MBTI (사주-MBTI 변환 엔진)

사주의 음양오행(Yin-Yang & Five Elements)과 십성(Ten Gods) 분석 기법을 바탕으로 개인의 성격 유형을 MBTI(E/I, S/N, T/F, J/P)의 네 가지 축으로 매핑하고, 가장 가능성이 높은 MBTI 유형을 추정하는 가설적 연구 모델입니다.

> [!NOTE]
> 본 프로젝트는 사주명리학과 MBTI 성격 유형 사이의 연관성을 탐색하기 위한 **학술적/가설적 매핑 모델**입니다. 과학적으로 검증된 성격 진단은 아니며, 사주팔자 상의 타고난 선천적 기질과 개인이 인지하는 MBTI 성향 사이의 상호작용을 비교 연구하는 용도로 권장됩니다.

---

## 🛠️ 주요 기능

1. **사주팔자 산출**: `sajupy` 엔진을 활용하여 입력한 출생 일시(양력)와 도시 정보를 기준으로 사주(연주, 월주, 일주, 시주)를 계산합니다.
2. **오행(Five Elements) 점수화**: 오행(목, 화, 토, 금, 수)의 분포와 강도를 계산합니다.
   - 사주 기둥별 위치 가중치(일주 1.4, 월주 1.5, 시주 1.1, 연주 0.8)와 천간/지지 가중치(천간 1.0, 지지 1.2)를 적용합니다.
   - 월지를 기준으로 태어난 계절의 기운에 대해 추가적인 계절 보정(20% 가중치 추가)을 적용합니다.
3. **십성(Ten Gods) 점수화**: 일간(Day Stem)을 기준으로 비겁, 식상, 재성, 관성, 인성의 십성을 분석하여 점수를 산출합니다.
4. **음양(Yin-Yang) 보정**: 사주 내 음양 비율을 산출하여 E/I 및 J/P 축의 판정에 보정치로 반영합니다.
5. **MBTI 4대 축 판정 및 강도 산출**: E/I, S/N, T/F, J/P 각 축에 대한 점수 및 선호도 강도(거의 동률, 약한 선호, 중간 선호, 강한 선호)를 도출합니다.
6. **최종 추정 MBTI 제안**: 가장 가능성 높은 1순위 유형과 점수 편차가 크지 않은 대안적 성향을 고려한 2순위, 3순위 추정치를 함께 제안합니다.
7. **실제 MBTI 비교**: 사용자의 실제 MBTI를 입력할 경우, 사주 추정 유형과 일치/불일치하는 축을 대조 분석하여 후천적 환경 요인의 영향력을 고찰할 수 있게 합니다.

## 🏗️ 가설적 매핑 규칙 개요

### 1. 위치 및 타입 가중치
- **위치 가중치**: 월주(1.5) > 일주(1.4) > 시주(1.1) > 연주(0.8)
- **타입 가중치**: 지지(1.2) > 천간(1.0)

### 2. MBTI 4대 축 매핑 기준
- **E / I (에너지 방향)**
  - **E (외향)**: 화(火) 기운, 목(木) 기운, 겁재, 식상, 편재, 양(陽) 기운 비율
  - **I (내향)**: 수(水) 기운, 금(金) 기운, 비견, 정관, 편인, 정인, 음(陰) 기운 비율
- **S / N (정보 인식)**
  - **S (감각)**: 토(土) 기운, 금(金) 기운, 식신, 재성, 관성
  - **N (직관)**: 목(木) 기운, 수(水) 기운, 화(火) 기운, 상관, 인성
- **T / F (판단 방식)**
  - **T (사고)**: 금(金) 기운, 수(水) 기운, 비겁, 재성, 편관, 정관
  - **F (감정)**: 화(火) 기운, 목(木) 기운, 토(土) 기운, 식신, 정인
- **J / P (생활 양식)**
  - **J (판단)**: 토(土) 기운, 금(金) 기운, 정재, 정관, 편관, 정인, 음(陰) 기운 비율
  - **P (인식)**: 목(木) 기운, 화(火) 기운, 수(水) 기운, 겁재, 식상, 편재, 편인, 양(陽) 기운 비율

## 🚀 시작하기

### 1. 패키지 설치
이 엔진은 `sajupy` 및 데이터 처리를 위해 `pandas`, `geopy` 등을 사용합니다. `requirements.txt`에 명시된 필수 패키지들을 설치하세요.

```bash
pip install -r requirements.txt
```

### 2. 테스트 실행
작동 여부를 검증하기 위한 2가지 대표 테스트 케이스가 작성되어 있습니다.

```bash
python test_engine.py
# (또는 사용 중인 파이썬 환경의 경로 직접 지정)
# C:\Users\may40\AppData\Local\Programs\Python\Python313\python.exe test_engine.py
```

### 3. 대화형 분석기 실행
인자를 전달하지 않고 `saju_mbti.py`를 실행하면 콘솔에서 사용자 입력을 직접 받는 대화형 분석 모드로 구동됩니다.

```bash

```

```bash

```
